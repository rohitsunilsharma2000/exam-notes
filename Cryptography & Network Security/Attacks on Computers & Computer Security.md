

### 1. What is DNS spoofing?

- **English:** DNS spoofing is a type of cyber attack where a hacker intercepts and alters DNS (Domain Name System) queries.  
  **Bangla:** DNS spoofing হলো একটি সাইবার আক্রমণের ধরন যেখানে একজন হ্যাকার DNS (ডোমেইন নাম সিস্টেম) কুয়েরি আটক করে এবং পরিবর্তন করে।

- **English:** It allows them to redirect users to fake websites or servers, potentially stealing sensitive information or installing malware.  
  **Bangla:** এটি হ্যাকারদের ব্যবহারকারীদেরকে জাল ওয়েবসাইট বা সার্ভারে পুনঃনির্দেশ করতে দেয়, যার ফলে সংবেদনশীল তথ্য চুরি বা ম্যালওয়্যার ইনস্টল করা যেতে পারে।

- **English:** DNS spoofing can be done through various means, including:  
  **Bangla:** DNS spoofing বিভিন্ন উপায়ে করা যেতে পারে, যার মধ্যে রয়েছে:

  - **English:** DNS cache poisoning: Hackers inject fake DNS records into a DNS cache, which is then served to users.  
    **Bangla:** DNS ক্যাশ বিষক্রিয়া: হ্যাকাররা DNS ক্যাশে মিথ্যা DNS রেকর্ড ইনজেক্ট করে, যা পরে ব্যবহারকারীদেরকে প্রদান করা হয়।
  
  - **English:** DNS tunneling: Hackers use DNS queries to tunnel through firewalls and gain unauthorized access.  
    **Bangla:** DNS টানেলিং: হ্যাকাররা DNS কুয়েরিগুলো ব্যবহার করে ফায়ারওয়ালের মধ্য দিয়ে টানেল তৈরি করে এবং অননুমোদিত প্রবেশাধিকার লাভ করে।
  
  - **English:** Man-in-the-middle (MITM) attacks: Hackers intercept DNS queries and alter them to redirect users to fake websites.  
    **Bangla:** ম্যান-ইন-দ্য-মিডল (MITM) আক্রমণ: হ্যাকাররা DNS কুয়েরি আটক করে এবং পরিবর্তন করে ব্যবহারকারীদেরকে জাল ওয়েবসাইটে পুনঃনির্দেশ করে।

### 2. What are the principles of security?

- **English:** The principles of security are the fundamental concepts that guide the design and implementation of secure systems.  
  **Bangla:** সুরক্ষার নীতিমালা হল মৌলিক ধারণা যা সুরক্ষিত সিস্টেমের নকশা ও বাস্তবায়ন পরিচালনা করে।

- **English:** The three main principles of security are:  
  **Bangla:** সুরক্ষার তিনটি প্রধান নীতি হলো:

  - **English:** Confidentiality: Protecting sensitive information from unauthorized access.  
    **Bangla:** গোপনীয়তা: অননুমোদিত প্রবেশাধিকার থেকে সংবেদনশীল তথ্য সুরক্ষিত রাখা।
  
  - **English:** Integrity: Ensuring that data is accurate, complete, and not modified without authorization.  
    **Bangla:** অখণ্ডতা: নিশ্চিত করা যে তথ্য সঠিক, সম্পূর্ণ এবং অননুমোদিতভাবে পরিবর্তিত হচ্ছে না।
  
  - **English:** Availability: Ensuring that systems and data are accessible and usable when needed.  
    **Bangla:** প্রাপ্যতা: নিশ্চিত করা যে সিস্টেম ও তথ্য প্রয়োজনে অ্যাক্সেসযোগ্য এবং ব্যবহারযোগ্য।

### 3. What is a DoS attack?

- **English:** A Denial of Service (DoS) attack is a type of cyber attack where an attacker attempts to make a computer or network resource unavailable by overwhelming it with traffic or requests.  
  **Bangla:** ডিনায়াল অব সার্ভিস (DoS) আক্রমণ হলো এমন এক প্রকার সাইবার আক্রমণ যেখানে আক্রমণকারী প্রচুর ট্রাফিক বা অনুরোধ পাঠিয়ে কম্পিউটার বা নেটওয়ার্ক রিসোর্সকে অপ্রাপ্য করে তোলার চেষ্টা করে।

- **English:** This can be done through various means, including:  
  **Bangla:** এটি বিভিন্ন উপায়ে করা যেতে পারে, যার মধ্যে রয়েছে:

  - **English:** Flooding: Sending a large amount of traffic to a network or system.  
    **Bangla:** ফ্লাডিং: একটি নেটওয়ার্ক বা সিস্টেমে প্রচুর পরিমাণে ট্রাফিক প্রেরণ।
  
  - **English:** Buffer overflow attacks: Sending more data than a system can handle, causing it to crash.  
    **Bangla:** বাফার ওভারফ্লো আক্রমণ: সিস্টেমের সামর্থ্যের বাইরে অতিরিক্ত ডেটা প্রেরণ, যার ফলে সিস্টেম ক্র্যাশ করে।
  
  - **English:** Application-layer attacks: Targeting specific applications or services to make them unavailable.  
    **Bangla:** অ্যাপ্লিকেশন-লেয়ার আক্রমণ: নির্দিষ্ট অ্যাপ্লিকেশন বা সার্ভিসগুলোকে লক্ষ্য করে সেগুলোকে অপ্রাপ্য করে তোলা।

### 4. What is IP sniffing and IP spoofing?

- **English:** IP sniffing: Intercepting and analyzing IP packets to gather information about a network or system.  
  **Bangla:** IP স্নিফিং: নেটওয়ার্ক বা সিস্টেম সম্পর্কে তথ্য সংগ্রহের জন্য IP প্যাকেটগুলো আটক করা এবং বিশ্লেষণ করা।

- **English:** IP spoofing: Sending IP packets with a fake source IP address to disguise the sender's identity.  
  **Bangla:** IP স্পুফিং: প্রেরকের পরিচয় লুকানোর জন্য মিথ্যা সোর্স IP ঠিকানা সহ IP প্যাকেট প্রেরণ করা।

### 6. What is the basic difference between a worm and a virus?

- **English:** A worm is a self-replicating malware that spreads without human interaction, while a virus requires human interaction (e.g., running an infected program) to spread.  
  **Bangla:** একটি ওয়ার্ম হলো এমন এক ধরনের ম্যালওয়্যার যা নিজে নিজে প্রতিলিপি নেয় এবং মানুষের হস্তক্ষেপ ছাড়াই ছড়িয়ে পড়ে, যেখানে ভাইরাস ছড়াতে মানুষের হস্তক্ষেপ (যেমন, সংক্রমিত প্রোগ্রাম চালানো) প্রয়োজন।

### 7. What is a brute-force attack?

- **English:** A brute-force attack involves systematically trying all possible combinations of passwords, keys, or encryption algorithms to gain unauthorized access.  
  **Bangla:** ব্রুট-ফোর্স আক্রমণে পাসওয়ার্ড, কী বা এনক্রিপশন অ্যালগরিদমের সকল সম্ভাব্য কম্বিনেশন সিস্টেমেটিক্যালি চেষ্টা করা হয় অননুমোদিত প্রবেশাধিকার অর্জনের জন্য।

- **English:** This can be done through various means, including:  
  **Bangla:** এটি বিভিন্ন উপায়ে করা যেতে পারে, যার মধ্যে রয়েছে:

  - **English:** Dictionary attacks: Using a list of words to guess passwords.  
    **Bangla:** ডিকশনারি আক্রমণ: পাসওয়ার্ড অনুমান করতে শব্দগুলির একটি তালিকা ব্যবহার করা।
  
  - **English:** Rainbow table attacks: Using precomputed tables of hash values to crack passwords.  
    **Bangla:** রেইনবো টেবিল আক্রমণ: পূর্বে গণনা করা হ্যাশ মানের টেবিল ব্যবহার করে পাসওয়ার্ড ভাঙা।
  
  - **English:** Exhaustive key searches: Trying all possible encryption keys to decrypt data.  
    **Bangla:** বিস্তৃত কী অনুসন্ধান: ডেটা ডিক্রিপ্ট করতে সকল সম্ভাব্য এনক্রিপশন কী চেষ্টা করা।

  **Bangla:** কম্পিউটার সিকিউরিটির মৌলিক উপাদানগুলোর মধ্যে রয়েছে:

  - **English:** Software: The programs and operating systems that run on hardware.  
    **Bangla:** সফটওয়্যার: সেই প্রোগ্রাম এবং অপারেটিং সিস্টেম যা হার্ডওয়্যারে চলে।

### 9. What is a threat and what are the categories of threats?

- **English:** A threat is a potential occurrence that could compromise security.  
  **Bangla:** একটি হুমকি হলো এমন সম্ভাব্য ঘটনা যা সুরক্ষা বিঘ্নিত করতে পারে।

- **English:** Threat categories include:  
  **Bangla:** হুমকির ধরনগুলোর মধ্যে রয়েছে:

  - **English:** Human threats: Unauthorized access, theft, or sabotage by individuals.  
    **Bangla:** মানবিক হুমকি: ব্যক্তিদের দ্বারা অননুমোদিত প্রবেশ, চুরি বা বিধ্বংস।
  
  - **English:** Natural threats: Natural disasters, such as earthquakes or hurricanes.  
    **Bangla:** প্রাকৃতিক হুমকি: প্রাকৃতিক দুর্যোগ, যেমন ভূমিকম্প বা ঘূর্ণিঝড়।
  
  - **English:** Environmental threats: Power failures, equipment failures, or other environmental factors.  
    **Bangla:** পরিবেশগত হুমকি: বিদ্যুৎ ব্যর্থতা, সরঞ্জামের ত্রুটি বা অন্যান্য পরিবেশগত কারণ।
  
  - **English:** Technological threats: Malware, hacking, or other technological vulnerabilities.  
    **Bangla:** প্রযুক্তিগত হুমকি: ম্যালওয়্যার, হ্যাকিং বা অন্যান্য প্রযুক্তিগত দুর্বলতা।

### 10. What are the security approaches?

- **English:** Security approaches include:  
  **Bangla:** সুরক্ষা প্রক্রিয়াগুলোর মধ্যে রয়েছে:

  - **English:** Preventive measures: Implementing security controls to prevent attacks, such as firewalls or access controls.  
    **Bangla:** প্রতিরোধমূলক ব্যবস্থা: ফায়ারওয়াল বা অ্যাক্সেস কন্ট্রোলের মতো সুরক্ষা নিয়ন্ত্রণ প্রয়োগ করে আক্রমণ প্রতিরোধ করা।
  
  - **English:** Detective measures: Implementing security controls to detect attacks, such as intrusion detection systems or logging.  
    **Bangla:** সনাক্তকরণ ব্যবস্থা: অনুপ্রবেশ সনাক্তকরণ সিস্টেম বা লগিং-এর মতো সুরক্ষা নিয়ন্ত্রণ প্রয়োগ করে আক্রমণ সনাক্ত করা।
  
  - **English:** Corrective measures: Implementing security controls to correct or mitigate the effects of an attack, such as backups or incident response plans.  
    **Bangla:** সংশোধনমূলক ব্যবস্থা: ব্যাকআপ বা ইনসিডেন্ট রেসপন্স প্ল্যানের মতো সুরক্ষা নিয়ন্ত্রণ প্রয়োগ করে আক্রমণের প্রভাব সংশোধন বা হ্রাস করা।

### 11. What do you mean by computer security and what are its goals?

- **English:** Computer security refers to the protection of computer systems, networks, and data from unauthorized access, use, disclosure, disruption, modification, or destruction.  
  **Bangla:** কম্পিউটার সিকিউরিটি বলতে বোঝায় কম্পিউটার সিস্টেম, নেটওয়ার্ক এবং ডেটাকে অননুমোদিত প্রবেশ, ব্যবহার, প্রকাশ, ব্যাঘাত, পরিবর্তন বা ধ্বংস থেকে সুরক্ষা দেওয়া।

- **English:** The goals of computer security are to ensure Confidentiality, Integrity, and Availability (CIA triad).  
  **Bangla:** কম্পিউটার সিকিউরিটির লক্ষ্য হলো গোপনীয়তা, অখণ্ডতা এবং প্রাপ্যতা (CIA ত্রিমূখ) নিশ্চিত করা।

### 14. What are the different types of attacks on computer and network systems?

- **English:** Types of attacks include:  
  **Bangla:** কম্পিউটার ও নেটওয়ার্ক সিস্টেমে আক্রমণের ধরনগুলোর মধ্যে রয়েছে:

  - **English:** Active attacks: Direct interference with a system or network, such as malware or DoS attacks.  
    **Bangla:** সক্রিয় আক্রমণ: সিস্টেম বা নেটওয়ার্কে সরাসরি হস্তক্ষেপ, যেমন ম্যালওয়্যার বা DoS আক্রমণ।
  
  - **English:** Passive attacks: Monitoring or eavesdropping on a system or network without interfering with it.  
    **Bangla:** নিষ্ক্রিয় আক্রমণ: সিস্টেম বা নেটওয়ার্কে হস্তক্ষেপ না করেই পর্যবেক্ষণ বা গুপ্তচরবৃত্তি করা।
  
  - **English:** Insider attacks: Attacks by individuals with authorized access to a system or network.  
    **Bangla:** ইনসাইডার আক্রমণ: সিস্টেম বা নেটওয়ার্কে অনুমোদিত প্রবেশাধিকার রাখেন এমন ব্যক্তিদের দ্বারা আক্রমণ।
  
  - **English:** Physical attacks: Attacks that involve physical access to a system or network, such as theft or sabotage.  
    **Bangla:** শারীরিক আক্রমণ: সিস্টেম বা নেটওয়ার্কে শারীরিকভাবে প্রবেশ করে চুরি বা বিধ্বংসের মতো আক্রমণ।

### 15. Distinguish between phishing and pharming. Why is it easier to fall prey to pharming than phishing?

- **English:** Phishing involves tricking users into revealing sensitive information, while pharming involves redirecting users to fake websites.  
  **Bangla:** ফিশিং হলো ব্যবহারকারীদের প্রতারিত করে সংবেদনশীল তথ্য প্রকাশ করানো, যেখানে ফার্মিং হলো ব্যবহারকারীদেরকে জাল ওয়েবসাইটে পুনঃনির্দেশ করা।

- **English:** Pharming is often easier to fall prey to because it doesn't require users to click on suspicious links or provide information.  
  **Bangla:** ফার্মিং-এ শিকার হওয়া প্রায়ই সহজ, কারণ এতে ব্যবহারকারীদের সন্দেহজনক লিঙ্কে ক্লিক করতে বা তথ্য প্রদান করতে হয় না।

### 16. What is the idea behind a man-in-the-middle attack?

- **English:** A man-in-the-middle (MITM) attack involves intercepting and altering communication between two parties, often to steal sensitive information or inject malware.  
  **Bangla:** ম্যান-ইন-দ্য-মিডল (MITM) আক্রমণের ধারণা হলো দুই পক্ষের মধ্যে যোগাযোগ আটক ও পরিবর্তন করা, প্রায়শই সংবেদনশীল তথ্য চুরি বা ম্যালওয়্যার ইনজেক্ট করার উদ্দেশ্যে।

---

### 17. Distinguish between active attacks and passive attacks.

- **English:** Active attacks involve direct interference with a system or network, while passive attacks involve monitoring or eavesdropping on a system or network without interfering with it.  
  **Bangla:** সক্রিয় আক্রমণগুলোতে সিস্টেম বা নেটওয়ার্কে সরাসরি হস্তক্ষেপ করা হয়, যেখানে নিষ্ক্রিয় আক্রমণে হস্তক্ষেপ না করেই পর্যবেক্ষণ বা গুপ্তচরবৃত্তি করা হয়।

---

### 18. What is access control? How is it different from availability?

- **English:** Access control refers to the mechanisms used to regulate who can access a system, network, or data.  
  **Bangla:** অ্যাক্সেস কন্ট্রোল হলো সেই পদ্ধতিসমূহ যা নির্ধারণ করে কে সিস্টেম, নেটওয়ার্ক বা ডেটাতে প্রবেশ করতে পারে।

- **English:** Availability refers to the ability of a system or network to provide services and data when needed.  
  **Bangla:** প্রাপ্যতা বলতে বোঝায় যে একটি সিস্টেম বা নেটওয়ার্ক প্রয়োজনে সার্ভিস ও ডেটা প্রদান করতে সক্ষম।

### 19. Cryptography: Concepts & Techniques

- **English:** Cryptography involves using algorithms and protocols to protect data from unauthorized access.  
  **Bangla:** ক্রিপ্টোগ্রাফি হলো এমন অ্যালগরিদম ও প্রোটোকল ব্যবহার করে ডেটাকে অননুমোদিত প্রবেশাধিকার থেকে সুরক্ষিত রাখা।

- **English:** Concepts and techniques include:  
  **Bangla:** ধারণা ও কৌশলগুলোর মধ্যে রয়েছে:

  - **English:** Encryption: Converting plaintext data into unreadable ciphertext.  
    **Bangla:** এনক্রিপশন: প্লেইনটেক্সট ডেটাকে এমন সাইফারটেক্সটে রূপান্তর করা যা পড়া যায় না।
  
  - **English:** Decryption: Converting ciphertext back into plaintext.  
    **Bangla:** ডিক্রিপশন: সাইফারটেক্সটকে পুনরায় প্লেইনটেক্সটে রূপান্তর করা।
  
  - **English:** Hashing: Creating a digital fingerprint of data to verify its integrity.  
    **Bangla:** হ্যাশিং: ডেটার অখণ্ডতা যাচাই করার জন্য একটি ডিজিটাল ফিঙ্গারপ্রিন্ট তৈরি করা।
  
  - **English:** Digital signatures: Using encryption and hashing to authenticate the sender of a message.  
    **Bangla:** ডিজিটাল সিগনেচার: এনক্রিপশন ও হ্যাশিং ব্যবহার করে বার্তার প্রেরককে প্রমাণীকরণ করা।
  
  - **English:** Key exchange protocols: Securely exchanging encryption keys between parties.  
    **Bangla:** কী এক্সচেঞ্জ প্রোটোকল: পক্ষগুলোর মধ্যে এনক্রিপশন কী সুরক্ষিতভাবে বিনিময় করা।
